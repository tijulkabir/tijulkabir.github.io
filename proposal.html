🍽️ Restaurant Management System

A modern system to streamline restaurant operations using Object-Oriented Programming principles.

💡 Key Features

  • Order Management: Dine-in & online ordering system.
  • Inventory Control: Stock tracking and expiration alerts.
  • Menu Management: Easy add/edit/remove menu items and prices.
  • User Roles: Separate interfaces for Customer, Staff, and Manager.

🧱 OOP Architecture

  • Encapsulation — Binding data and operations.
  • Inheritance — Shared User class extended by specific roles.
  • Polymorphism — Same interface, different behaviors.
  • Abstraction — Simplified interface hiding complexity.

📈 Impact

The system reduces time and human error while boosting efficiency across operations.
